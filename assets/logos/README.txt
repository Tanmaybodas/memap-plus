Logo Assets for MeMap+

This directory contains platform logos for the enhanced graph visualization.

Supported platforms:
- github.png: GitHub logo (green theme)
- reddit.png: Reddit logo (orange theme) 
- instagram.png: Instagram logo (purple theme)

The application will gracefully fall back to colored dots if logo files are not present.

Logo files should be:
- PNG format
- Square aspect ratio (recommended: 64x64 or 128x128 pixels)
- Transparent background preferred
- Optimized for web display
